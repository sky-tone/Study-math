## 2026-04-08 — v1.3 答案自动匹配

### 新增功能

#### 答案匹配工具 (`update_answers.py`)
- 从 `GZMK数学答案.pdf` 提取 26 套试卷的答案文本
- 自动解析答案结构：试卷名、题型段落、题号→答案映射
- 支持批量选择题（"1-5：ADBCB"）、判断题（×/√）等格式
- 基于源文件编号题目索引 + 文本相似度匹配导入题目
- 按 (试卷名, 题型, 题号) 精准查找答案

| 指标 | 数量 |
|------|:----:|
| 答案来源试卷 | 26 套 |
| GZMK来源题目 | 670 |
| 成功匹配 | 637（95.1%）|
| 未匹配（OCR乱码等） | 33 |
| 非GZMK来源（无答案源）| 195 |

### 技术细节
- 解析 4 个源文件（试卷1/2 的 txt + ocr 版本），建立完整编号题目索引
- 处理全角句号（．）、孤儿题目（文件开头无标题的题）等边缘情况
- 支持判断题 ×/√ 特殊字符答案
- 验证通过：大奥填空1=49、填空3=8、填空6=99、解答5=700 等

## 2026-04-06 — v1.1 题库清洗 + Web 版上线

### 新增功能

#### 独立 Web 应用 `app.py`（954行）
- 使用 Streamlit 构建完整 Web 界面，无需 VS Code 即可使用
- 7 个功能页面：知识树总览、学习知识点、公式速查表、测验系统、错题本、学情分析、题目编辑器
- 题目编辑器：支持在线编辑/新增/删除题目，修改答案和解析
- 溯源功能：可定位题目在原始 OCR/PDF 文本中的位置
- 启动方式：`python -m streamlit run app.py`

#### 桌面版应用 `desktop_app.py`
- 封装为独立桌面窗口，双击 `run_desktop.bat` 即可运行
- 使用 Edge/Chrome App 模式（无地址栏/标签页，外观如原生应用）
- 后台自动启动/停止 Streamlit 服务器
- 关闭窗口后自动退出，无残留进程
- 桌面模式自动隐藏 Streamlit 部署按钮和菜单栏
- 无需额外安装任何依赖（Edge 预装于 Windows 10/11）

### 题库清洗

- 对 `imported_exercises.json` 进行质量清洗，移除 63 道低质量/重复/OCR 识别错误的题目
- 清洗前：562 题 → 清洗后：499 题
- 主要清洗内容：
  - 数的分类：60 → 44（-16，移除大量 OCR 乱码题）
  - 比和比例的基本概念：20 → 15（-5）
  - 水费的分段计费问题：10 → 6（-4）
  - 其他知识点各移除 1-3 道低质量题
  - 移除空知识点「可能性的大小」（仅 1 道残缺题）
- 清洗后题库质量显著提升，合并后练习题从 510 降至 454 道
- 有练习的知识点从 41 个变为 40 个（新增 14 个暂无练习）

## 2026-03-27 — v1.0 完整系统上线

### 新增功能

#### 核心知识库 `knowledge_base.py`（1978行）
- 构建完整知识体系：4章 → 14小节 → 54知识点
- 编写 54 个概念讲解（含 Markdown/LaTeX 公式和重点列表）
- 内置 36 道精选练习题（含答案和详细解析）
- 实现动态加载机制：启动时自动合并 `data/imported_exercises.json`
- 提供 6 个辅助函数：`get_all_topics()`, `get_concept()`, `get_exercises_by_topic()` 等

#### 自动导入管线 `import_questions.py`（555行）
- 题目解析器：按题号从 OCR/PDF 文本中拆分独立题目
- 知识点分类器：54 个知识点的关键词映射表，支持正则匹配
- 难度评估器：基于文本长度、子问题数、关键词综合评分（1-3级）
- 题型推断器：自动识别选择/填空/判断/计算/解答
- ID 生成器：MD5 哈希确保唯一性
- JSON 合并保存：自动去重，增量更新
- 命令行支持：单文件/批量/PDF直接导入

#### OCR 提取工具 `ocr_extract.py`（196行）
- 使用 RapidOCR 对扫描件 PDF 逐页 OCR
- 支持自定义 DPI（默认300）
- 置信度过滤（>0.5）
- 逐页进度显示
- 快速预览模式

#### 教学系统 `teaching_learn.ipynb`
- 知识树总览（54知识点索引表，含概念/练习标记）
- 按编号选择知识点，展示概念讲解 + 配套练习
- 答案与解析查看
- 公式速查表（平面图形/立体图形/统计/行程 全套公式）

#### 测验系统 `quiz_test.ipynb`
- 两种模式：综合随机测验 / 专项知识点测验
- 动态组卷：随机抽题，题目数量可配置
- 自动批改：字符串匹配，支持大小写和空格容忍
- 错题本：按知识点分组，自动记录不重复
- 学情分析：总体统计、薄弱知识点排行、历次记录、进步趋势
- 学习记录持久化到 `data/study_record.json`

### 题库数据

- OCR 提取 GZMK 数学试卷1（95页扫描件，504秒完成）→ 260 题
- OCR 提取 GZMK 数学试卷2（11页扫描件，56秒完成）→ 50 题
- PDF 提取小学数学总复习2022（86页文字PDF）→ 252 题
- 自动导入合并后总计 **510** 道练习题，覆盖 **41** 个知识点

### Bug 修复

- 修复 `knowledge_base.py` 中文引号 `""` 导致的 SyntaxError
- 修复 `ocr_extract.py` emoji 字符在 Windows GBK 终端导致的 UnicodeEncodeError
- 修复 `ocr_extract.py` RapidOCR 返回字符串类型 confidence 导致的 TypeError
- 修复 `quiz_test.ipynb` 答案列表 `my_answers` 硬编码 5 项导致的题目数量不匹配
- 修复 `quiz_test.ipynb` 跳过组卷直接运行批改时的 NameError
- 修复 `quiz_test.ipynb` 多次测验 `records` 数据覆盖问题
- 安装缺失的 `ipykernel` 和 `rapidocr-onnxruntime` 到 `.venv`

### 开发环境

- Python 3.14.2
- 虚拟环境 `.venv`
- 依赖：pymupdf, rapidocr-onnxruntime, ipykernel, numpy, opencv-python, onnxruntime, Pillow
