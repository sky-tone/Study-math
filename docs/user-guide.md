# 小升初数学学习系统 — 使用指南

> 欢迎使用！本系统帮助你系统复习小学数学，通过"学知识 → 做练习 → 测一测 → 看错题"的方式，高效备战小升初。

---

## 准备工作

### 第一次使用

1. 用 **VS Code** 打开 `Study-math` 文件夹
2. 如果右下角弹出提示选择 Python 解释器，选择 `.venv` 里的那个
3. 打开任意 `.ipynb` 文件，如果提示选择内核，选择 `.venv (Python 3.x)`

### 什么是 Notebook？

Notebook（.ipynb 文件）由一个个"格子"组成，每个格子可以**单独运行**：
- 点击格子左边的 **▶ 播放按钮**，或按 **Ctrl+Enter** 运行当前格子
- 格子必须**从上往下按顺序运行**（第一次使用时）

## 一、学习知识点（teaching_learn.ipynb）

打开 `teaching_learn.ipynb`，按以下步骤操作：

### 步骤 1：初始化

运行**第 1 个代码格子**（点击 ▶），看到 `✅ 知识库加载成功！` 即可。

### 步骤 2：查看知识树

运行**第 2 个代码格子**，会显示完整的知识点列表：

```
| 编号 | 知识点           | 概念 | 练习  |
|:----:|-----------------|:----:|:-----:|
|  1   | 数的分类         |  📖  | 📝×3  |
|  2   | 数的组成         |  📖  | 📝×5  |
| ...  | ...             |  ... |  ...  |
```

- 📖 = 有概念讲解
- 📝×N = 有 N 道练习题

### 步骤 3：选择知识点学习

找到**第 3 个代码格子**，修改 `TOPIC_NUM` 的数字：

```python
TOPIC_NUM = 1  # ← 改成你想学的编号，比如 5
```

然后运行这个格子，系统会显示：
- **概念讲解**：包含公式、定义、例题
- **重点记忆**：需要牢记的要点
- **配套练习**：相关习题（先自己做！）

### 步骤 4：查看答案

做完练习后，运行**第 4 个代码格子**，查看每道题的答案和详细解析。

### 步骤 5：公式速查（可选）

运行**最后一个代码格子**，显示所有重要公式的速查表（面积、体积、行程等）。

### 💡 小贴士

- 想换一个知识点？回到步骤 3，改 `TOPIC_NUM` 的数字，重新运行即可
- 不需要重新运行第 1、2 步

## 二、测验检测（quiz_test.ipynb）

打开 `quiz_test.ipynb`，按以下步骤操作：

### 步骤 1：初始化

运行**第 1 个代码格子**，看到：
```
✅ 测验系统已就绪！
📊 历史记录：已完成 X 次测验
```

### 步骤 2：查看可选知识点

运行**第 2 个代码格子**，显示所有可以测验的知识点和题目数量。

### 步骤 3：选择测验模式

找到**第 3 个代码格子**，修改两个参数：

```python
TEST_MODE = 0      # 0 = 综合随机（所有知识点混合）
                   # 或填编号，如 3 = 只测"质数与合数"
NUM_QUESTIONS = 5  # 每次出几道题
```

运行后，系统会随机出题，显示在屏幕上。

### 步骤 4：做题并提交答案

1. **先在纸上做题**，写下你的答案
2. 找到**第 4 个代码格子**，第一次运行会看到提示：

```
✏️ 请在 my_answers 列表中填写 5 个答案：
   my_answers[0] = ""  # 第1题
   my_answers[1] = ""  # 第2题
   ...
```

3. 在格子代码中找到 `my_answers`，把答案填进去：

```python
my_answers[0] = 'A'          # 选择题写选项
my_answers[1] = '72'         # 填空题写答案
my_answers[2] = 'x = 5'      # 计算题写结果
my_answers[3] = '3570'       # 数值题写数字
my_answers[4] = '对'          # 判断题写"对"或"错"
```

4. **再次运行这个格子**，系统自动批改，显示：
   - ✅ 正确的题目
   - ❌ 错误的题目（含正确答案和解析）
   - 📊 成绩单和薄弱知识点提示

### 步骤 5：查看错题本

运行**第 5 个代码格子**，查看你做错过的所有题目，按知识点分组：

```
📌 百分数的概念（2题）
   题目: ...
   正确答案: ...
   解析: ...
```

### 步骤 6：查看学情分析

运行**第 6 个代码格子**，查看：
- 总体统计（测验次数、正确率、平均分）
- 薄弱知识点排行（出错最多的知识点）
- 历次测验记录
- 进步趋势

### 步骤 7：清空错题本（可选）

如果你确定已经掌握了所有错题，可以在**最后一个代码格子**中取消注释来清空。

## 常见问题

### Q: 运行格子后没有反应？
确保右上角选择了正确的 Python 内核（`.venv`），然后**从第一个代码格子开始**按顺序运行。

### Q: 公式显示不正常？
确保使用 VS Code 打开 Notebook，VS Code 自带 LaTeX 渲染支持。

### Q: 怎么知道我学了多少？
打开 `quiz_test.ipynb`，运行"学情分析"格子，可以看到完整的学习统计。

### Q: 可以增加新题目吗？
可以！把新的试卷 PDF 放到项目目录下，然后在终端运行：
```bash
python ocr_extract.py "新试卷.pdf"          # 如果是扫描件
python import_questions.py "data/新试卷_ocr.txt"
```
下次打开 Notebook 或 Web 版时新题目自动可用，不需要改代码。

## 六、桌面版使用（推荐！）

最简单的使用方式，双击即可运行，看起来跟普通软件一样：

### 方式一：双击启动（最简单）

双击项目根目录下的 `run_desktop.bat` 文件即可。

### 方式二：命令行启动

```bash
python desktop_app.py
```

### 桌面版特点

- 自动用 Edge/Chrome 打开独立窗口（无地址栏、无标签页）
- 外观跟普通桌面软件一样
- 关闭窗口后自动退出，无残留进程
- 不需要安装任何额外软件

### 功能页面

| 页面 | 功能 |
|------|------|
| 🌳 知识树总览 | 浏览完整知识体系，查看各知识点的概念和练习覆盖情况 |
| 📖 学习知识点 | 选择知识点，查看概念讲解 + 配套练习 |
| 🧮 公式速查表 | 平面图形/立体图形/统计/行程全套公式 |
| 🎯 测验系统 | 综合/专项测验，自动组卷和批改 |
| 📕 错题本 | 按知识点分组查看错题 |
| 📈 学情分析 | 总体统计、薄弱知识点排行、进步趋势 |
| ✏️ 题目编辑器 | 在线编辑/新增/删除题目，修改答案和解析 |
